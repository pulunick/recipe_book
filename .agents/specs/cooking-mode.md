# 쿠킹 모드 (Cooking Mode) 기능 명세

## 개요

실제 요리 중 스마트폰을 보면서 사용하는 전용 화면.
기존 상세 페이지와 별개로 동작하며, 요리에 집중할 수 있는 UX를 제공한다.

## 진입/종료

- **진입**: `/my-recipes/[id]` 상세 페이지 상단 [쿠킹 모드 시작] 버튼
- **URL**: `/my-recipes/[id]/cook` (별도 라우트)
- **종료**: 상단 X 버튼 or 뒤로가기 → 상세 페이지 복귀

## 화면 구성

### 헤더 (고정)
```
[X 종료]        3단계 / 7단계        [진행바 ████░░░]
```

### 메인 영역 (전체화면)
```
┌─────────────────────────────────┐
│                                 │
│  3                              │  ← 단계 번호 (대형, 배경)
│                                 │
│  팬에 기름을 두르고              │  ← 단계 설명 (1.8rem+)
│  센불로 달군 뒤                  │
│  돼지고기를 넣습니다.            │
│                                 │
│  ┌────────────────────────┐     │
│  │  ⏱  2분  [시작]        │     │  ← 타이머 (있을 때만)
│  └────────────────────────┘     │
│                                 │
└─────────────────────────────────┘

     [← 이전]          [다음 →]
```

### 완료 화면
```
✓ 완성!
오늘도 맛있게 드세요 🍳

[요리 횟수 +1 기록하기]
[레시피로 돌아가기]
```

## 주요 기능

### Wake Lock API
- 쿠킹 모드 진입 시 화면 꺼짐 방지
- 종료 시 해제
- 미지원 브라우저 자동 fallback (무시)

```typescript
let wakeLock: WakeLockSentinel | null = null;
onMount(async () => {
  if ('wakeLock' in navigator) {
    wakeLock = await navigator.wakeLock.request('screen');
  }
});
onDestroy(() => wakeLock?.release());
```

### 타이머
- `step.timer` 있을 때만 표시
- [시작] 클릭 → 카운트다운 진행
- 0초 도달 시 진동 or 알림음 (가능한 경우)
- 단계 이동 시 타이머 리셋

### 네비게이션
- 버튼 클릭 or 좌우 스와이프
- 첫 단계에서 이전 비활성화
- 마지막 단계에서 다음 → 완료 화면

### 다크 테마
- 기본: 어두운 배경 (`#1a1a1a`) + 흰 텍스트
- 주방 조명 환경에서 눈부심 방지

## 기술 스택

- SvelteKit 별도 라우트: `src/routes/my-recipes/[id]/cook/+page.svelte`
- 기존 load function에서 동일한 CollectionItem 데이터 재사용
- Svelte 네이티브 트랜지션으로 단계 전환 애니메이션

## 우선순위 / 향후 과제
