## 1. ChefChef (https://www.chefchef.app/ko)

### 핵심 기능
- YouTube URL → AI 레시피 자동 추출
- **음성 제어**: "다음 단계", "타이머 시작" 음성 명령
- 1,000개+ 레시피 데이터베이스 (채널별 검색)
- 다국어 번역 (한/영/중/일)
- 커뮤니티 댓글, iOS 앱 출시

### 수익 모델 (Freemium)
| | 무료 | 프리미엄 |
|--|------|---------|
| 추출 대기 | **30분** | 5분 |
| 음성 조작 | ❌ | ✅ |
| 실시간 AI Q&A | ❌ | ✅ |
| 광고 | 있음 | 없음 |

14일 무료 체험 제공.

### 특이사항
- 프리미엄 없이는 30분 대기 → 실사용 장벽 높음
- 재료 저장/장보기 기능 없음
- 개인화 기능 부족 (즐겨찾기만)

## 2. 해먹으리 (https://www.hemogry.com)

### 핵심 기능
- 유튜브·인스타그램 영상 → AI 레시피 변환 (17개국 사용)
- 타임스탬프 기반 장면 이동
- AI 어시스턴트 (칼로리, 대체 재료 질문)
- **장보기 기능**: "Add All to Shopping List" 한 번에 추가, AI 자동 정리

### 특이사항
- **경쟁사 중 유일하게 장보기 기능 보유**
- 레시피 포크 (개인화된 저장)
- 식자재 구매 기능 (외부 연동 가능성)
- 하단 탭: 홈 / 북마크 / 검색 / 장바구니

## 3. 쉐프토리 (https://www.cheftories.com/ko)

### 핵심 기능
- YouTube 링크 → AI 단계별 레시피 생성
- **음성 제어**: "다음 단계", "타이머 3분" (음성 키워드: "토리아")
- **자동 타이머**: 단계별 권장 시간 자동 설정
- **장면 검색**: "양파 썰기" 등 영상 내 특정 장면 즉시 이동
- iOS / Android 앱 출시, 다국어 지원 (한/영/일/중)

### 수익 모델
- 기본 무료, 프리미엄 모델 미공개

### 특이사항
- 음성 제어 + 자동 타이머 가장 강력
- 재료 저장/장보기 없음
- 개인화 기능 거의 없음 (즐겨찾기만)

## 해먹당 포지셔닝

> **"AI가 추출한 레시피를 진짜 내 것으로 만드는 개인 레시피북"**

### 핵심 차별점
1. **완전 무료 + 즉시** — 경쟁사 대기 없음, 유료 없음
2. **가장 강력한 개인화** — 태그, 별점, 메모, 요리 이력, 맛 프로필 (경쟁사 전무)
3. **레시피 메타 정보 자동 추출** — 인분, 소요 시간, 난이도를 AI가 자동 추출 (경쟁사 미지원 또는 수동 입력)
4. **장보기 기능** (개발 예정) — 해먹으리만 보유, 해먹당도 추가 시 차별화
5. **쿠킹 모드** (개발 예정) — 음성 없이도 주방에서 편하게

## 향후 논의 필요 항목

### 🤖 AI 기능 확장
- [ ] **요리 도우미 AI** — 재료 대체 제안 (꿀 → 올리고당, 우유 → 두유 등), 알레르기 대체, 냉장고 재료로 대체 추천
- [ ] **레시피 편집 기능** — 내 보관함 레시피를 직접 수정 (재료 양, 단계 설명 변경 등) / DB 단 변경 필요 (user_collections에 recipe_override 컬럼 추가 방향 검토)
- [ ] **영양 정보 자동 추출** — 칼로리, 단백질, 탄수화물 등 AI 추정값 제공

### 🍳 쿠킹 모드 고도화
- [ ] **기본 쿠킹 모드** — 단계별 전체화면, Wake Lock, 타이머 (1순위, 명세 작성 완료)
- [ ] **유튜브 영상 연계** — 레시피 단계 클릭 시 해당 유튜브 타임스탬프 구간으로 바로 이동
- [ ] **음성 제어** — "다음 단계", "타이머 시작" 음성 명령 (쉐프토리 대응, 고도화 단계)

### 🛒 장보기 기능
- [ ] **장보기 리스트** — 레시피 재료를 한 번에 쇼핑 리스트에 추가 (보관함 목록 페이지에서도 이용 가능하게)
- [ ] **멀티 레시피 장보기** — 여러 레시피의 재료를 합산한 통합 쇼핑 리스트
- [ ] **외부 마트 연동** — 쿠팡이츠, 마켓컬리 등 외부 서비스 연동 (장기)

### 📱 플랫폼 / 배포
- [ ] PWA / 모바일 앱 전환 시점
- [ ] 수익화 모델 방향 (완전 무료 유지 vs 프리미엄)
