# 해먹당 업데이트 로그

---

## v0.4.0 — 2026-03-04

### 새 기능

- **원본 영상 카드**: 레시피 페이지 하단의 "원본 영상 보기 →" 텍스트 링크가 유튜브 썸네일을 활용한 카드로 교체됨. 채널명과 영상 원제목 표시, hover 시 애니메이션 효과
- **재료 미리보기**: 레시피 목록 카드에서 "재료 N개" 버튼으로 재료 목록을 접고 펼 수 있음. 최대 5개 표시 후 "외 N개" 표시
- **맨 위로 버튼**: 레시피 상세 페이지에서 스크롤을 300px 이상 내리면 우측 하단에 "↑" 버튼 표시

### 개선

- **태그 삭제**: 레시피 목록의 태그 뱃지에서 직접 × 버튼으로 태그 제거 가능
- **별점 표시 전용**: 레시피 목록 카드의 별점은 보기 전용으로 변경. 별점 매기기는 상세 페이지에서만 가능

### 버그 수정

- **태그 팝오버 클리핑**: 태그 추가 팝오버가 카드 경계에 잘려 보이던 문제 수정
- **태그 팝오버 방향**: 팝오버가 위로 열리던 것을 아래로 열리도록 변경
- **태그 팝오버 링크 차단**: 태그 팝오버가 열린 상태에서 다른 레시피 카드를 클릭하면 이동이 안 되던 문제 수정. 이제 팝오버 영역 외부 클릭 시 팝오버가 닫히고 링크도 정상 동작
- **AI 재료 양 표현**: 소금, 후추 등 조미료의 양이 null로 표시되던 문제 수정. 이제 "약간", "1작은술" 등 적절한 표현으로 표시됨

## v0.3.0 — 2026-02-xx

### 새 기능

- **레시피 저장**: 분석된 레시피를 "내 레시피북에 추가" 버튼으로 저장
- **태그 시스템**: 레시피에 컬러 태그 생성 및 부착. 사이드바에서 태그별 필터링
- **별점 & 요리 횟수**: 상세 페이지에서 별점 1~5점 설정, "오늘 요리했어요 🍳" 버튼으로 요리 횟수 기록
- **즐겨찾기**: 레시피 카드 썸네일의 ⭐ 버튼으로 즐겨찾기 추가/해제
- **단골 레시피 뱃지**: 요리 횟수 3회 이상 시 "단골 레시피 🏆" 뱃지 표시
- **메모**: 상세 페이지에서 나만의 메모 추가 및 수정
- **카테고리 자동 분류**: AI가 레시피를 한식/양식/중식 등 15개 카테고리로 자동 분류

### 개선

- **레시피 타이틀**: AI가 순수 요리명만 추출 (예: "마파두부"), 채널명은 별도 표시
- **결과 페이지 분리**: 분석 결과가 `/recipe/[id]` 별도 페이지로 이동. 홈 화면은 항상 깨끗하게 유지
- **만드는 법 타임라인**: 단계별 원형 배지 + 완료 체크 기능. 완료된 단계는 흐리게 처리

## v0.2.0 — 2026-01-xx

### 새 기능

- **내 레시피북**: 저장한 레시피 목록 페이지 (`/my-recipes`)
- **레시피 상세**: 저장된 레시피 상세 페이지 (`/my-recipes/[id]`)
- **재료 체크리스트**: 상세 페이지에서 재료를 체크하면 로컬에 저장
- **맛 프로필 차트**: 짠맛/단맛/매운맛/신맛/기름짐 5가지 축 레이더 차트
- **모바일 sticky 버튼**: 모바일에서 하단 고정 "레시피북에 추가" 버튼
