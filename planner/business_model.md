# 비즈니스 모델 (BM) 전략

> 핵심 기능(YouTube 영상 멀티모달 분석 → 구조화 레시피 JSON + 개인화 데이터 축적)을 기반으로 한 상업적 확장 전략

---

## 1. B2C 구독형 모델 (Freemium)

유저가 직접 체감하는 핵심 가치를 기반으로 부분 유료화 적용.

### 무료 플랜 (Free)
- 월 5~10회 YouTube URL 분석 제공
- 기본 레시피 변환 및 저장소 기능

### 프리미엄 플랜 (Pro — 예: 월 4,900원)
- **무제한 분석**: URL 분석 횟수 제한 해제
- **냉장고 연동 및 맞춤 쇼핑**: 레시피에서 부족한 재료 추출, 유통기한 관리 연동 (PRD 3단계 확장 기능)
- **고도화된 개인화**: "사용자님의 평소 취향보다 더 맵습니다. 고춧가루를 1스푼 줄이세요" 등 AI 맞춤형 팁 및 비율 자동 재계산

---

## 2. 커머스 제휴 마케팅 ★ 가장 현실적이고 강력한 초기 수익원

AI가 영상에서 뽑아낸 **정확한 재료 리스트와 수량**이라는 강점을 극대화.

### 식재료 원클릭 구매 연결
- 쿠팡 프레시, 마켓컬리, SSG, B마트 등과 파트너스 API 제휴
- 레시피 재료 리스트 하단에 **[내 냉장고에 없는 재료 한 번에 장바구니 담기]** 버튼 제공
- 유저가 해당 링크를 통해 구매 시 **판매 수수료 약 3~5%** 취득

### 조리 도구 추천
- 조리 과정에서 특정 도구 감지 시 (예: 에어프라이어) 해당 기기 구매 링크 자연스럽게 노출

---

## 3. B2B / 크리에이터 타겟 SaaS 모델

일반 유저뿐 아니라 콘텐츠 번역/가공이 필요한 비즈니스로 확장.

### 요리 유튜버 & 인플루언서 타겟
- 유튜버가 영상 URL을 올리면 YouTube '더보기' 란이나 블로그에 올릴 수 있는 완벽한 레시피 텍스트/HTML 자동 생성
- 다국어 번역 확장 시 글로벌 유튜버의 고민(해외 시청자를 위한 정확한 레시피 자막/설명 작성) 해결
- 월정액 과금

### API 판매 (건당 과금)
- 다이어트 앱, 식단 관리 앱, 스마트 냉장고 제조사(삼성, LG 등)에
  **"YouTube URL → 구조화 레시피 JSON 반환"** API 자체를 건당 과금으로 판매

---

## 4. 고도화된 타겟 광고 (Data & Ads)

앱 내에 쌓이는 유저의 '입맛 데이터'와 '냉장고 데이터'의 높은 광고 가치 활용.

### 초개인화 식음료 타겟 광고
- '매운맛 지수 5'를 자주 저장하는 유저에게 특정 라면 브랜드/매운 소스 신제품 맞춤 광고 노출
- 단순 배너가 아닌 실제 **취향(Flavor) 데이터 기반** 광고 → 광고 전환율(CVR) 높아 단가 높음

---

## 5. 권장 전개 방향

```
[Phase 1 — 런칭 직후]
무료 서비스
  + 쿠팡 파트너스/마켓컬리 장보기 연동 (제휴 마케팅)
  → 자연스러운 첫 수익으로 서버 비용(Vercel, Render 등) 충당

[Phase 2 — 잔존율 확보 후]
유저 데이터(레시피 저장, 맞춤 지표) 축적 확인
  → 구독형 프리미엄 기능 도입
     (냉장고 연계, 비율/취향 자동 보정)
  → B2B API 판매 병행
```
